# Movie Showcase Website

A personal movie archive and cinematic showcase for movies I have watched, currently designed around a sleek black-and-white-only aesthetic.

Each movie will eventually have its own page or section designed around that movie's specific aesthetic, mood, genre, and visual identity while staying strictly within the black-and-white direction. Variation should come from contrast, typography, imagery, layout, texture, and motion rather than hue-based themes.

## Project Status

This repository currently contains:

- A detailed development plan in `MOVIE_SHOWCASE_PLAN.md`
- A small deployable starter page in `index.html`

## Deployment

This project is prepared for GitHub Pages deployment from the `main` branch.

## Planned Site Areas

- Home page
- Movie gallery
- Individual movie pages
- Genre pages
- Favorites
- Recently watched
- About page

## Long-Term Goal

Create a personal cinematic archive where each movie feels like its own interactive monochrome poster and mini-world.
