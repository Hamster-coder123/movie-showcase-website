# Movie Showcase Website

A personal movie archive and cinematic showcase for movies I have watched.

Each movie will eventually have its own page or section designed around that movie's specific aesthetic, mood, colors, genre, and visual identity.

## Project Status

This repository currently contains:

- A detailed development plan in `MOVIE_SHOWCASE_PLAN.md`
- A small deployable starter page in `index.html`

## Deployment

This project is prepared for GitHub Pages deployment from the `main` branch.

## Planned Site Areas

- Home page
- Movie gallery
- Individual movie pages
- Genre pages
- Favorites
- Recently watched
- About page

## Long-Term Goal

Create a personal cinematic archive where each movie feels like its own interactive poster and mini-world.
